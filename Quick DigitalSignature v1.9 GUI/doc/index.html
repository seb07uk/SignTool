Quick DigitalSignature GUI

UsageUżytkowanie

How to Use

Jak używać

Signing a File

Podpisywanie pliku

1

Select file

Wybierz plik

Click Browse… or drag-and-drop the target binary onto the window.

Kliknij Browse… lub przeciągnij i upuść plik na okno aplikacji.

2

Choose certificate

Wybierz certyfikat

Select a .pfx from the dropdown or click Browse… to locate one manually.

Wybierz .pfx z listy lub kliknij Browse…, aby zlokalizować ręcznie.

3

Enter password

Wpisz hasło

Enter the PFX password (leave blank if none). Check Save to store it securely.

Wpisz hasło PFX (zostaw puste jeśli brak). Zaznacz Zapisz dla bezpiecznego przechowywania.

4

Sign

Podpisz

Click Sign File. A confirmation dialog appears with the SignTool output on success.

Kliknij Podpisz plik. Po sukcesie pojawi się okno z wynikiem SignTool.

TroubleshootingRozwiązywanie problemów

Troubleshooting

Rozwiązywanie problemów

⚠ SignTool.exe not found

⚠ Nie znaleziono SignTool.exe

Place signtool.exe next to the application, or install the Windows SDK and add its bin\<version>d\ path to your system PATH.

Umieść signtool.exe obok aplikacji lub zainstaluj Windows SDK i dodaj ścieżkę bin\<wersja>d\ do PATH.

⚠ Verification fails — untrusted root certificate

⚠ Weryfikacja nie powiodła się — niezaufany certyfikat

Use the Install Root CA and Install Intermediate CA buttons. Alternatively, use PowerShell:

Użyj przycisków Zainstaluj Root CA i Zainstaluj Intermediate CA, lub PowerShell:

Import-Certificate -FilePath root.cer -CertStoreLocation Cert:\LocalMachine\Root
Import-Certificate -FilePath inter.cer -CertStoreLocation Cert:\LocalMachine\CA

⚠ Save password is greyed out

⚠ Opcja zapisu hasła jest wyszarzona

Windows DPAPI is unavailable for this user session. Enter the password manually each run.

Windows DPAPI niedostępne dla tej sesji. Wpisuj hasło ręcznie przy każdym uruchomieniu.

⚠ Certificate not in the list

⚠ Certyfikat nie na liście

Click the Refresh (🔄) button, or use Browse… to locate the .pfx file manually.

Kliknij Odśwież (🔄) lub użyj Browse…, aby ręcznie wskazać plik .pfx.
